For Developers:

To generate the HTML documentation files from XSLT:

  1. Install xsltproc (or use an online XSLT processor)
  2. Run: xsltproc -o Documentation.html documentation.xsl Documentation.xml
  3. Run: xsltproc -o TestDocumentation.html test-documentation.xsl TestDocumentation.xml
  4. Commit the generated HTML files to the repository
